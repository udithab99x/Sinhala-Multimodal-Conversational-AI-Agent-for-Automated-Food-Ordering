# Sinhala Multimodal Conversational AI Agent for Automated Food Ordering

**Advanced AI вҖ” Final Group Project**  
A production-ready Sinhala AI assistant that handles food orders through **phone calls** and **WhatsApp** using a fine-tuned Gemma 4 LLM.

---

## AI Techniques Implemented

| Technique | Where |
|-----------|-------|
| **NLP** вҖ” intent classification, entity extraction, text preprocessing | `notebooks/04_nlp_and_rag.ipynb`, `app/agent.py` |
| **Transformer / LLM** вҖ” Gemma 4 E2B fine-tuned with LoRA | `notebooks/02_fine_tuning_gemma4.ipynb` |
| **Prompt Engineering** вҖ” zero-shot, few-shot, chain-of-thought | `notebooks/03_prompt_engineering.ipynb` |
| **RAG** вҖ” FAISS + sentence-transformers menu retrieval | `app/rag.py`, `notebooks/04_nlp_and_rag.ipynb` |

## Setup

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Configure credentials

```bash
cp .env.example .env
# Edit .env вҖ” add your Twilio and HuggingFace credentials
```

### 3. Fine-tune the model (Lightning AI or Google Colab)

- Open `notebooks/02_fine_tuning_gemma4.ipynb` on **Lightning AI** (free T4 GPU)
- Run all cells вҖ” model trains and pushes to your HuggingFace repo
- Copy the HuggingFace repo ID into `.env` вҶ’ `HF_MODEL_ID`

### 4. Run the app

```bash
# Start FastAPI server
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload

# In another terminal, expose it publicly (for Twilio webhooks)
ngrok http 8000
```

### 5. Configure Twilio

**WhatsApp Sandbox:**
1. Go to Twilio Console вҶ’ Messaging вҶ’ Try it out вҶ’ Send a WhatsApp message
2. Set webhook URL: `https://your-ngrok-url.ngrok.io/whatsapp/incoming`

**Voice (phone call):**
1. Go to Twilio Console вҶ’ Phone Numbers вҶ’ your number
2. Set webhook URL: `https://your-ngrok-url.ngrok.io/voice/incoming`

### 6. Terminal demo (no Twilio / no Google Cloud needed)

```bash
python app/demo_terminal.py
```

## Model: Gemma 4 E2B

- **Base:** `google/gemma-4-e2b-it` (2B edge model, multilingual, 140+ languages)
- **Fine-tuning:** LoRA (rank 16), 4-bit quantization via Unsloth
- **VRAM required:** ~4 GB (4-bit) вҖ” fits on Lightning AI free T4 GPU
- **Inference:** CPU-compatible after quantization (Ollama / llama.cpp)

## Dataset

- **15 synthetic Sinhala food-ordering conversations** (hand-crafted)
- Covers: formal Sinhala, informal Sinhala, Sinhala-English mixed, slang
- Intents: food_order, confirm, cancel, menu_inquiry, delivery_inquiry, etc.
- Entities: food items, quantities, spice levels, addresses, payment methods
