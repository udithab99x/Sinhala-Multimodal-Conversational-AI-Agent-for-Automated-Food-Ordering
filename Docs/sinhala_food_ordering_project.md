# Sinhala Multimodal Conversational AI Food Ordering Assistant

## Project Overview
Build a Sinhala-speaking AI assistant that can handle food orders through both voice calls and WhatsApp messaging. The system should understand customer requests, ask follow-up questions, confirm orders, and forward finalized orders to restaurants automatically.

The same AI model should power both channels so the experience stays consistent across voice and chat.

## Objectives
- Develop a Sinhala conversational AI for food ordering.
- Support both telephone calls and WhatsApp conversations.
- Fine-tune a large language model using Sinhala food-ordering conversations.
- Implement NLP techniques for intent recognition and entity extraction.
- Integrate Retrieval-Augmented Generation (RAG) for menus and pricing.
- Evaluate fine-tuning against baseline models.
- Deploy on low-resource infrastructure without a dedicated GPU.

## Core Features

### Voice Ordering
- Customer calls a phone number.
- AI answers in Sinhala.
- AI understands spoken Sinhala.
- AI collects order details.
- AI confirms order details.
- AI forwards completed orders.

### WhatsApp Ordering
